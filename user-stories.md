# User Stories

* As a user, I want to navigate between the app's different views so that I can easily access Top Movies, All Movies, and Add Movie.

* As a user, I want to view a curated list of Top Movies so that I can browse highly recommended films.

* As a user, I want to browse all movies stored in the database so that I can view my complete movie collection.

* As a user, I want to search the movie collection by title so that I can quickly find a specific movie.

* As a user, I want to sort the movie collection alphabetically so that I can organize the list from A to Z.

* As a user, I want to submit a form with a title, genre, rating, image URL, and description so that I can add a new movie to my collection.

* As a user, I want the newly added movie to appear immediately in my collection without refreshing the page.

* As a user, I want to delete a movie from my collection so that I can keep my movie list up to date.
